Atlas token architecture
4-tier token model
Core → raw values (hex, px, ms)
Semantic → intent-based aliases (action, feedback, neutral…)
Component → per-component tokens
Density → comfortable / compact variants
Deprecated tokens ⊘
Secondary brand (blues/purples) and Gray family are deprecated.
Use Neutral tokens instead. ⊘ icons in Figma mark deprecated values.
